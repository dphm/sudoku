Sudoku
======

3x3 Sudoku puzzle and simple recursive solver

Input
=====
A textfile with the following format:

0 0 6 0 3 0 0 0 0
0 0 0 2 0 1 8 0 5
0 0 8 7 5 4 0 2 0
8 4 0 3 0 0 0 5 0
0 0 9 0 0 2 1 0 0
0 0 2 9 0 0 7 0 6
0 9 0 0 4 0 3 6 0
0 6 5 0 0 7 0 0 4
7 8 4 0 0 0 0 0 9

A blank square is represented by a 0.

Output
======
A solved puzzle, if it is possible, or false if it is impossible.
